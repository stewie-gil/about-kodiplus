About KodiPlus

KodiPlus is a revolutionary web application designed to simplify the process of finding suitable and affordable rental properties. Inspired by personal experiences of struggling with rental searches, the KodiPlus team set out to create a user-centric platform that addresses the challenges faced by lower and middle-income individuals in Kenya. This project is a Portfolio Project for Holberton School.
